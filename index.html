🌐 Setting Up Your Professional Portfolio

A Step-by-Step Guide for SI Analytics 30-Day Data Visualization Challenge Participants

🧭 Overview

This guide walks you through creating a professional data-science portfolio website using GitHub Pages and Lovable.ai. By the end, you’ll have a personal website showcasing your data visualization projects, CV, and social profiles — all free and publicly accessible.

🪄 Step 1: Set Up Your GitHub Account

  1. Visit https://github.com → click Sign up.
  2. Follow the instructions to create and verify your account.
  3. Once logged in, you’re ready to create your first repository.
💡 GitHub is one of the most generous and reliable platforms for hosting static websites — ideal for professional portfolios.

🗂️ Step 2: Create a New Repository

  1. Click the “＋” icon (top-right corner) → New repository.
  2. Name it portfolio (or website).
  3. Keep visibility Public.
  4. Click Create repository.

📁 Step 3: Upload Your Portfolio Files

  1. Click on your new repository name (e.g., portfolio).
  2. Select Add file → Upload files.
  3. Upload the following:
    • Your CV (PDF or Markdown)
    • A professional headshot
    • Screenshots of dashboards or visualizations
    • Logo (optional)
  4. Click Commit changes to save your uploads.
  5. To get each file’s link:
    • Open the file → right-click → Copy address (URL).

💻 Step 4: Create Your Portfolio Website on Lovable.ai

  1. Go to Lovable.ai Invite Link → Sign up.
  2. Click Create Project → name it eportfolio or any preferred name.
  3. Paste the following prompt into the text area:
Design a modern, elegant portfolio website for a data science and analytics professional participating in the SI Analytics 30-Day Data Visualization Challenge.

The website should look clean, credible, and visually data-driven, optimized for both desktop and mobile.

🔹 Core Sections
1. Home / About Me:
   A short professional bio introducing the participant — include name, title, and summary (e.g., “Data Scientist passionate about transforming data into insight and impact”).
   Optionally include a professional headshot and logo (from GitHub Pages).

2. My Projects:
   Showcase selected data visualization works from the 30-day challenge — link screenshots, dashboards, GitHub repositories, and interactive notebooks.
   Each project card should have: title, tool used (R / Python / Power BI), description, and “View Project” link.

3. Other Platforms:
   Add icons + links to:
   - GitHub
   - LinkedIn
   - Google Scholar
   - X (Twitter)
   - Facebook (optional)

4. CV / Resume Section:
   Embed or link directly to your hosted CV on GitHub Pages (PDF or Markdown).

5. Contact:
   Include a simple contact form or “Let’s Collaborate” button.

✨ Visual & UX Style
• Use subtle data-themed icons or animated metrics.
• Add hover effects on project cards.
• Keep transitions fluid (fade or slide-in).
• Include footer: “Made with ❤️ by an SI Analytics Participant.”

🧠 Optional Enhancements
• Add a timeline or badge gallery of challenge days.
• Include testimonials or mentor feedback.

📸 Assets to Include
Insert GitHub or Drive links for:
• Headshot(s)
• Dashboard screenshots
• CV (PDF link)
• Logo

💡 Creative Tip
If you admire an existing website’s layout, paste its link or screenshot for Lovable.ai to mirror its style.

⏳ Step 5: Wait and Publish

  1. Wait a few minutes while Lovable.ai generates your site.
  2. Free accounts start with 10 credits and receive 5 new credits daily.
  3. When satisfied with your preview, click Publish.
  4. You’ll see a message confirming success and get your public link (e.g., https://yourname.lovable.app).
🎉 Congratulations! You now have a professional data-science portfolio website.

🔗 Step 6: Share Your Portfolio

  • Post your link on LinkedIn, X (Twitter), or Facebook using hashtags:
    #SIAnalytics #DataVisualization #Portfolio #DataScience #AIForGood
  • Add it to your CV, email signature, or GitHub bio.
  • Record a short screen demo (1–2 minutes) showing how you built it.
